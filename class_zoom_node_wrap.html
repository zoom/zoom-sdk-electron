static

Initialize ZOOM SDK.

Parameters
1.path(string) On win os is the sdk.dll path. On mac os is empty.
2.domain(string) Web domain.
3.langid(number) The ID of the SDK language. see ZNSDK_LANGUAGE_ID enum.
4.langName(string) Resource name.
5.langInfo(string) The value should be the full path of the resource file when the langType value is CustomizedLanguage_FilePath, including the file name. When the langType value is CustomizedLanguage_Content, the value saves the content of the resource.
6.strSupportUrl(string) Support URL.
7.enable_log(boolean) Enable log feature.
Returns
If the function succeeds, the return value is ZNSDKERR_SUCCESS. Otherwise failed. To get extended error information, see ZNSDKError enum.
