Note: Both ZoomAudioDevice.driver and ZoomAudioDevice.kext is used for support share computer audio in share.
Usage: Copy this two plugins to YourAPPName/Contents/Plugins folder path.
ZoomAudioDevice.driver used for Mac OS version >= 10.10
ZoomAudioDevice.kext used or Mac OS version <= 10.9
If you uninstall app,this also need to uninstall.
